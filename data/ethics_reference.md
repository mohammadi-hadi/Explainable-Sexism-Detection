# Data Ethics Reference - Chapter 4

## Data Source

### EXIST 2023 Shared Task Dataset
- **Provider**: EXIST 2023 at CLEF/IberLEF
- **Description**: Dataset for sexism identification in social media
- **Languages**: English and Spanish
- **Access**: Through shared task registration

## Ethics Considerations

- Data was collected and provided by EXIST 2023 organizers
- Used under terms of the shared task
- No additional data collection performed
- Results presented at aggregate level

## Original Study Ethics

The EXIST 2023 dataset was created under the ethical guidelines of the organizing institutions. This study performs analysis using the provided benchmark data.

## Usage

Data used solely for research purposes as permitted by the shared task terms.
